#《与小卡特一起学Python》1
##推荐序-陈俊
作者说在自己孩子8岁时教他编程，只为培养他专一求精的精神。
现代孩子的通病:什么都玩一下，兴趣广泛，但对哪个都不通。不通非常致命，因为不通孩子就无法建立自信，没有自信就很容易躲避困难，然后又更不自信，如此恶性循环。没有自信作为基础，孩子很难发挥自己的能力，形成自己的特长。所以说从小能有一个特长，有一个专注的、足够突出的拿手项目，对孩子的发展非常重要。
至于学编程会不会变成IT民工命，我想这其实是个”伪命题“。现在少年MBA、少年领导力、情商培训很火，不是说它们没有作用，但管理能力和情商在成长的过程中可以潜移默化地培养，而具备一项专业的技能才是立身之本。
##前言
编程(programming)就是告诉计算机要做什么。
为什么学编程

  • 不论作为业余爱好还是作为职业，编程都会很有意思，都会让你很有收获。
  • 如果你对计算机感兴趣，想更多地了解它到底是怎么工作的，想知道怎们才能让它做你想做的事情。
  • 如今计算机已经无处不在，工作中、学习中、生活中都黄有可能使用计算机。学习编程能让你从总体上更好地了解计算机。
    为什么选用Python语言
  • 最初创建Python语言的出发点就是为了便于学习。Python程序是最易读、最容易编写，也是最容易理解的。
  • Python是免费的，可以下载到很多用Python编写的模块。
  • Python是开源软件，并且可以在各种不同的计算机上运行。
  • Python非常适合学习编程，全世界每天都有成千上万的这也人士在使用这种语言。包括NASA、Google这些机构的程序员。
    怎样使用本书
  • **验证例子:**通过自己动手编程你可以学到的东西多得多。
  • **输入程序:**通过亲手输入程序，培养对编程的一种感觉。
  • **做习题:**尽可能多地多做些习题，做题之前千万别看答案，除非你实在做不出来了。

##1 基础知识
IDLE交互模式
从IDLE启动Python，IDLE是一个Python shell，可以通过输入文本与程序交互。在交互模式中，只需键入变量名就可以显示这个变量的值，而不必使用print。
语法错误告警
THere’s an error in your program:invalid syntax.
变量命名规则:

  • 可以有字母、数字和下划线（_）；
  • 必须以字母或下划线开头，不能以数字开头；
  • 变量名区分大小写；
  • 变量名中不能有空格。
    长字符串
    三重引号字符串，可以得到一个跨多行的字符串。
    变量赋值
    程序可以在任何时间对变量重新赋值(把标签贴在新东西上)。
    **Tips: **编程中最常见的bug就是改变了不该改变的变量，或者尽管改变的变量无误，但是改变的时机不对。

int() 从一个字符串或浮点数创建一个新的整数，实际上是去掉小数部分；
str() 从一个数创建一个新的字符串；

##函数
要把程序分成较小的部分，主要有三种方法:函数、对象、模块。
函数是可以完成某个功能的代码块，就像是代码的积木，可以反复使用。
def中的代码并不是主程序的一部分，所以程序运行时会跳过这一部分。函数调用时才运行函数里的代码，如果定义一个函数而不调用它，函数里的代码永远也不会运行。
**传参:**使用多个参数时，要用逗号来分隔。如果函数有超过5到6个参数，应该考虑把所有参数集中到一个列表中，然后把这个列表传递到函数。
变量作用域
对于函数而言，函数内的名字只是在函数运行时才会创建。Python在函数运行时会创建新的名字在函数内使用，当函数完成时会把他们删除。在函数运行结束时，其中的所有名字都不存在。
如果在函数中使用主程序中定义的变量名，Python允许你使用这个变量，但是如果函数的任何部分试图改变这个变量，Python会创建一个新的局部变量。

##对象
对象可以把函数和数据收集在一起，把程序中的各部分描述为自包含的单元。
一个对象的特征成为属性，动作(能够对对象做的操作)称为方法
对象的创建分为两步:第一步，定义类(勾画蓝图，并不会真的创建一个对象)；第二步，使用类来创立一个对象(实例化)。
属性并不属于类，它们属于各个实例。
**初始化对象:**创建类定义时，可以定义一个特殊的方法，init()，只要创建这个类的一个新实例，就会运行这个方法，还可以向该方法传递参数。
特殊方法:str()告诉Python打印(print)一个对象时具体显示什么内容。默认显示

  • 实例在那里定义
  • 类名
  • 存储实例的内存位置
    如果你希望print为对象显示其他内容，可以定义自己的__str__()，它会覆盖内置的__str__()方法。
    **self:**self参数会告诉方法哪个对象调用它(实例引用)
    **代码桩pass:**可以使用pass关键字作为一个占位符。
