Linux学习记录

Linux再经典、再强大，充其量也只是个操作系统，是一个工具。虽说“磨刀不误砍柴工”，但是一定要对自己所要砍的“柴”有清醒的认识，不能盲目地在工具上投入过多的经历，在耽于磨刀的过程中失去方向，还用所谓的“不误”来为自己的踌躇不前辩解。

##1、初识Linux
2008年6月临近大学毕业的时候，我有幸收到一位同学的推荐邀请，注册加入hdchina，接触到PT的文件分享机制，下载收藏了一批高清析度的视频和音乐。《Revolution.OS.2001.DVDRip.XviD-RETRO.avi》是其中的一个，影片内容是关于计算机操作系统中Windows和Linux两大阵营之间的竞争。通过观看这部影片，我了解到Linux是一种完全不同于Windows的操作系统——Linux自创立之初就是开放源代码源、支持自由软件的。
第一次成功安装Linux操作系统已经是2011年2月时的事情了。当时在单位同事的建议下将ASUS笔记本操作系统更换为Win7（之前一直用WindowsXP），整理完台式机的硬盘数据后，台式机空闲出来，在上面尝试安装了Ubuntu操作系统，看过几个教学视频后，没有更深入地去学习。之后的七年中除了使用安卓手机外，与Linux无任何交集。
（没有找到好的教程和学习材料或许是没有继续学习Linux的原因之一。单位周边的第一个3G基站是2011年2~3月份才建立起来的，更早些的时候工作单位不仅没有互联网，也没有手机3G网络信号，几个玩魔兽世界网友的同事都是在用中国移动的2G网络，数据传输只能靠GPRS。）
##2、Ubuntu下配置深度学习框架
2016年ALphaGo与李世乭的人机大战掀起了机器学习、深度学习的热潮，我于17年10月开始学习开始学习Python（一门在容易上手的编程语言，在机器学习领域拥有较高的占有率），18年初加入一个深度学习的QQ小组。深度学习对于电脑性能有着更高的要求，老旧的笔记本电脑已经不能胜任这一需求，在和妻子商议后，花费了￥7000+在京东上购买配件自己组装了台式电脑。在安装Win7的过程中总是不能正确地识别驱动键盘，于是在同事的推荐下安装了Win10。

根据组长安排的学习进度，我们在3月份集中两个星期的时间学习Linux操作系统，要求完成深度学习框架的配置。当时推荐的学习材料除了网易的公开课之外还有《鸟哥的Linux私房菜基础学习篇-第三版》。我下载ubuntu-16.04.4-desktop-amd64的版本，安装在台式机单独的一块硬盘中，经过多次的尝试成功地配置好GPU版本的tensorflow框架。但是自5月2日运行完“猫狗大战”的例子之后，没有继续投入时间和精力在深度学习方面。
##3、CentOS的学习
深度学习告一段落，但是Sophie组长推荐的电子书却给我留下了深刻的印象。之后我曾在IPad上断续的阅读该书的部分章节，并于18年6月购买了《鸟哥的Linux私房菜：服务器架设篇（第三版）》纸质书。学习《服务器架设》的过程中发现基础篇已经更新到第四版了，与第三版不同，基础篇的第四版已经改用CentOS7进行讲解。
2018年12月下载CentOS-7-x86_64-DVD-1810版本后，前后数次在ASUS笔记本电脑以及新台式机上安装CentOS7，并进行了一些操作练习。19年1月购买《基础学习篇（第四版）》纸质书，学习的同时开始做笔记记录一些要点。目前已学完前面九章，最后面的四个章章也粗略的看过一遍。
###实践及收获
参照教材第三章内容在ASUS笔记本电脑上安装了Linux和Win7双系统，实现了多重开机安装流程与管理（grub2）。几经周折安装好了shadowsocks-qt5客户端，对于Linux下的软件安装机制有了一定得认识。Linux水平自评——（初级）： 了解系统的启动流程及开机中的引导 ；基本了解Linux的文件系统及目录架构；知道Linux下文件及用户的权限；会在Linux对磁盘进行分区，格式化，挂载以及设置启动挂载。
##4、一些思考
18年12月重新启动CentOS的学习后，自己的思路不是很清晰，目标也不够明确。
对于笔记本、台式机以及硬盘资源的规划一直犹豫不决。一方面想一次到位，把Linux打造成自己顺手的工具；另一方面又舍弃不了用顺手了的Windows。安装、格式化、再次安装，来来回回折腾了几次，浪费了大量的时间。最后才定下来在旧笔记本上安装双系统，用于CentOS学习过程中的实践以及移动办公；台式机保持Win10主用操作系统，用于处理日常的其他事项。
Linux再经典、再强大，充其量也只是个操作系统，是一个工具。虽说“磨刀不误砍柴工”，但是一定要对自己所要砍的“柴”有清醒的认识，不能盲目地在工具上投入过多的经历，在耽于磨刀的过程中失去方向，还用所谓的“不误”来为自己的踌躇不前辩解。

今天，你读书了吗？

不管你假装和掩饰的技术有多么娴熟 ，你的真正的需求 、价值观 、信仰和感觉都会源源不断地流露出来 ，你所传授的只是你自己的样子 ——不多也不少 。

18年4月上旬，阅读《如何阅读一本书》的过程中，萌生了总结梳理并分享自己读书经历的想法。为了促成这一想法化为实际行动，我主动向领导请缨，提出要在世界读书日时在单位做了一场题为“这些年，我的读书生活”的汇报。历时10天，共23个小时的准备后（包含一次小范围试讲），最终于4月24日正式做了汇报，实现自己的构想。去年的汇报共分六个部分，现简要整理如下。
##1、我的读书经历
此部分回顾读书的经历，划分为：初高中启蒙阶段，04-09年的入门阶段，10-16年的中级进阶阶段，17年之后的中高级阶段。具体每个阶段的经历之后会单独成文，这里仅列出每个阶段自己读的代表性书目，主要用意在于说明我几乎是大学毕业工作之后才慢慢地走上读书的道路，给入门的读者一个具体的参考案例。
###大学前的启蒙
小学《西游记连环画》；初中《福尔摩斯探案集》、《三国演义》（约105回）；高中《我与地坛》、《命若琴弦》、《红楼梦》（约40回）
###入门阶段
大学04-08：《王牌战舰的覆灭》、《射雕英雄传》、《神雕侠侣》；08-09：《蒋百里传》、《麦克阿瑟回忆录》、《货币战争1-2》、《笑傲江湖》
###中级进阶
10：《少有人走的路》、《家》、《潜规则》、《八十忆双亲·师友杂忆》、《国史大纲引论》、《中国历史研究法》、《妞妞》、《嫌疑人X的献身》
11-13：《暗时间》、《学会提问》、《谁会认错》、《不能承受的生命之轻》、《雨季不再来》、《让我们始终牵手旅行》
14-16：《时间简史》、《宝宝也是哲学家》、《让孩子的大脑自由》、《魔法岁月》、《翻转课堂的可汗学院》、《高效能家庭的7个习惯》、《稀缺》、《奇特的一生》
###中高级
17：《故事》、《人类简史》、《认识商业》、《知识大融通》、《战争论》、《国富论》、《亲密关系》
18：《大脑的未来》、《如何阅读一本书》、《何谓文化》、跟大师学语文系列

##2、读书的收获
其次总结了读书的收获，整体的感觉有两个：一、比原著更精彩的影视作品是屈指可数的；二、优秀的书籍总是避不开历史和哲学。具体来说是增进了对世界的理解和认识、应用于实际生活改变了自己的行为，最重要的收获是自己对待时间的态度有了根本性的转变。
##3、为什么要读书
第三粗略论述为什么要读书。其一，读书可以获取成长所需的间接经验；其二，真正的阅读可以升华精神层面的需求；其三，读书是自我教育的必经之路；最后，读书可以为孩子树立榜样，通过家庭教育来帮助学校教育。
##4、我可以读书吗
第四部分“读书，该如何开始”，主要是解答初级入门读书时常见的疑惑，比如：“文化水平低，读书适合我吗？”“现在才开始读书是不是太晚了？”、“我没时间读书”、“如何面对周围人异样的眼光”、“我追不上别人”、“开卷有益吗？”等，帮助入门读者做好思想上的准备。
##5、选什么书
第五部分就选什么书的问题做了简单的介绍，对于入门级的读书者，建议根据自己感兴趣的点去选书，可以是感兴趣的电影、电视剧的原著，对棋类感兴趣的可以看看小说《棋王》，对武侠剧感兴趣的可以看看金庸的短篇小说《越女剑》。之后可以尝试阅读一些散文集，小说等文学作品，之后可以阅读一些任务传记，再高阶一些的可以读读历史和哲学。
##6、如何阅读一本书
最后结合《How to Read a Book》（怎样阅读一本书）讲解如何阅读一本书，着重强调阅读一本书需要问的四个问题：

(1)整体来说，这本书到底在谈些什么？（找出这本书的主题）；(2)作者细部说了什么，怎么说的？（找出主要的想法、声明与论点）；(3)本书说得有道理吗？是全部，还是部分？（知道作者的想法是不够的，做出自己的判断）；(4) 这本书跟你有什么关系？

Shadowsocks 客户端设置教程 - Linux

Shadowsocks 客户端设置教程 - Linux

Shadowsocks 设置方法 (Linux)

  • GUI 客户端 Shadowsocks-QT5
  • 命令行客户端

Shadowosocks-QT5

下面以 Ubuntu 64 位系统做参考，其他系统请参考 安装指南

安装 Shadowsocks-Qt5

按照下面的说明在 Ubuntu 上下载并安装 Shadowsocks-Qt5。

1. 下载客户端

转到 Shadowsocks 下载页面。 点击最新版本的 Shadowsocks-Qt5-x.x.x-x86\_64.AppImage 进行下载 (x.x.x为版本号) 。

2. 安装客户端

右键单击 “Shadowsocks-Qt5-x.x.x-x86_64.AppImage” > 选择 “Properties” > 单击 “Permissions” > 在”Allow excuting file as program” 上打勾。

配置 Shadowsocks 账号

登陆我们的网站, 执行下列操作：

  • 单击 “服务” > “我的服务” > 选择 “你可用的产品/服务”。
  • 查看你的节点信息。

在您的电脑上， 执行下列操作：

  • 双击Shadowsocks-Qt5-x.x.x-x86\_64.AppImage > “Connection” > “Add” > “Manually”。

  • 填写 “Server Address” （为你的节点服务器地址）> 填写”Server Port”（为你的服务端口) > 填写 “Password” （为你的登陆密码)。
  • 在 “Local Address” 处填上 “127.0.0.1” > 在 “local Port” 处填上 “1080” > 选择 “Encryption Method” （为你的加密方式)。

勾选 “Auto connect to connect on application start”, 将会在开启 Shadowsocks-QT5 时自动连接至该服务器。

您也可以通过二维码方式单独增加节点，在您的计算机上， 执行下列操作：

此二维码同样适用于其他客户端。

  • 双击Shadowsocks-Qt5-x.x.x-x86\_64.AppImage > “Connection” > “Add” > “Scan QR code on Screen” > 当弹出新的 “Profile Editor”后，点击 “OK”。
  • 点击 “Connect” 图标 。

命令行客户端

1. 安装：

Python : https://github.com/shadowsocks/shadowsocks/tree/master#install
Shadowsocks-libev: https://github.com/shadowsocks/shadowsocks-libev#installation

下面我们以 Python 版的 Shadowsocks 为例

安装命令：
Debian / Ubuntu:

apt-get install python-pip

pip install git+https://github.com/shadowsocks/shadowsocks.git@master

CentOS:

yum install python-setuptools && easy\_install pip

pip install git+https://github.com/shadowsocks/shadowsocks.git@master

For CentOS 7, if you need AEAD ciphers, you need install libsodium

dnf install libsodium python34-pip

pip3 install git+https://github.com/shadowsocks/shadowsocks.git@master

Linux distributions with snap:

snap install shadowsocks

2. 创建 Shadowsocks 配置文件

创建一个 /etc/shadowsocks.json 文件，格式如下

{

"server":"服务器 IP 或是域名",

"server\_port":端口号,

"local\_address": "127.0.0.1",

"local\_port":1080,

"password":"密码",

"timeout":300,

"method":"加密方式 (chacha20-ietf-poly1305 / aes-256-cfb)",

"fast\_open": false

}

3、启动 Shadowsocks

Python 版客户端命令是 sslocal ， Shadowsocks-libev 客户端命令为 ss-local

/usr/local/bin/sslocal -c /etc/shadowsocks.json -d start

4、终端内使用，需安裝 proxychains

Debian / Ubuntu：

sudo apt-get install proxychains

编辑 /etc/proxychains.conf

修改最后一行

socks5 127.0.0.1 1080

接着我们就可以直接 用 proxychains + 命令的方式使用代理，例如

proxychains curl xxxx

proxychains wget xxxx

sudo proxychains apt-get xxxx

5、其他程序使用

需要在程序内设置一个协议为 socks v5 的代理

服务器：127.0.0.1

端口：1080 #应与Shadowsocks客户端的本地端口对应，默认为1080

6、 关闭 Shadowsocks

在终端内输入

lsof –i:1080

kill 相应的 pid 即可

2019年初学英语

在前一篇文章中，通过回顾20年英语学习经历，认清了自我英语水平仍处于很低层次的现实，分析了提升英语水平的必要性，展望了熟练掌握英语的光明前景，同时也立下决心——“于2019年开启英语学习的新征程，计划在5年内稳步提升自我英语水平”。本文是行动两个月以来的复盘分析。

##时间记录
起初预定的目标是每天至少投入1个小时。两个月以来实际投入62小时，其中一月份38小时47分钟，二月份23小时38分钟，具体行动时间段记录如下图所示。

虽然整体平均下来勉强达到每天1个小时，但是显而易见的是行动的持续性、稳定性非常差。一月份中22天有行动，缺席9天；最长的连续行动10天，最长间断2天。二月份中13天有行动，缺席多达15天；最长连续行动3天，最长间断4天。

##学习内容
###词汇基础训练营
1月5日开始参加“英语词汇基础训练营”在线课程，间隔一天上课，课程所用材料是BBC LearningEnglish，直播上课时间是晚上21:00-22:30。听完第一次直播课后，决定后面的课通过听回放录音来学习，这样既可以相对自由的安排学习时间，又能避免在线听课时因网络问题造成的卡顿，获得更好的学习体验。
讲课时老师会重点讲解学习材料中英文单词在上下文中的英文释义，作业要求是每次课程后以语音形式打卡5个单词或短语，另外老师强烈建议朗读课上所学的材料（包括话剧中的对话和旁白）。

及时听完课程11次，补课5次；
完成单词打卡11次，缺席4次；
完成朗读9次，缺席6次。
###TED演讲练口语

只听了一次直播课，其他的课程都是后来补听的；没有参与过课堂发言，课后作业一次也没做！
间隔了三个星期才开始朗读课程中的材料。
##感悟体会
###英语认知的再次刷新
参加的两个课程讲授者是即将进入日内瓦大学攻读翻译博士的英语同声传译员William老师。不得不说Willam老师刷新了我对英语学习的认知，使我认识到英国和美国的日常使用的英语并不是我们教材中见到的那些句子。英语的学习不在于大词难词的掌握，小词的运用，介词、冠词的运用都至关重要。采用英文释义的词典才能更好地理解一个单词的含义，一次只需记住单词在上下文中的那一个释义，不必急于急于求成地查看一个单词的所有含义。
###朗读
杨振宁曾在清华大学教授《大学物理》课程，第一次课的开场白中他提到朗读对于英语语感的培养至关重要，建议学生每天朗读15分钟以上。
这两个月的学习中我朗读了部分材料，也发现了不少之前读错的发音。基本上20秒的语音需要练习8~10分钟才可以读到自己基本满意，语音语调方面的模仿上距离其他同学还有很大的差距。
###同行的小伙伴
在18年12月再次朗读新概念一课文后，开始构想19年的英语学习。今年行动伊始，偶遇3名有英语学习意向的朋友。
在我的建议下，基础相对好些的L也一起加入了“基础词汇训练营”的学习。现在去看看“小鹅通”中他也是只完成了11次的语音单词打卡，其他方面的学习情况不太清楚。
D因为听说将来可能无法参加CET4考试，热情已退去大半。想来当初找我的时候他是最热情澎湃的，什么“不管以后学得怎么样，我真的十分感谢您……”等等的话都出来了。我当时头脑还是十分清醒的，赶紧让他打住，因为两年多的实践下来，我很清楚持续行动地去做一件事情是多么的不容易。“善始者实繁，克终者盖寡”是亘古不变道理。
三人中的Z是我当初最看好的一个，感觉上他是那种愿意下笨功夫的人。目前应该还在持续地投入时间和精力学习英语（待核实）。
春节恰好是在这个时间段中间，虽然节前也跟他们仨强调“假期适当娱乐，学习英语的行动不要断”（我对他们的建议是每周至少保证4天的行动量），但实际结果似乎并没有我想象中的好。（10天前提议让他们每个人总结下两个月以来的行动，截至现在只有D简单总结了几句话。）
虽然只是4个人的小团队，不去认真经营的话实际的效果也是会大打折扣的。明知道他们在行动的过程中，热情极有可能会逐渐消退，没有更好地鼓励或者激励他们持之以恒地行动——这是我的责任，也是我的遗憾。

2019春节记事

不知不觉中七天长假已过完了。今年春节年味儿非常淡，若不是陪孩子们玩玩小烟花，我几乎都感觉不到是在过年。
#一、感觉不到年味儿的原因
首先，在单位过年与在家中过年的反差很大。14年休年假，15年、16年、17年因为在学校读研有寒假，18年春节休假也休了年假，连续五年春节都是在家里过的。今年没能回家过年，和前面五年形成了鲜明的对比。没有家人团聚的温馨，少了走亲访友的热闹，见不到儿时熟悉的村庄景象，年味可能也就无从谈起。
其次，两年以来持续不断的学习行动后，我对节假日已几乎免疫。成长的道路上是没有节假日的，正如俗话所说“马不吃夜草不肥”。如果你在某个方面存在不足，跟别人比起来有差距，那么节假日正是你实现“弯道超车”最好的时机。在其他人沉浸于节假日氛围狂欢之时，你每天学上一两个小时，差距可能就会缩小点儿。此外，节假日还是最好的补旧账的机会。如果你一直都在持续行动的道路上前进，几乎不可避免地会因为工作或者家庭等原因没能完成某一天的计划，这时节假日让你有相对充足的时间用来弥补之前欠下的债。
#二、乏善可陈的春节生活
有了前两年的经验，今年制定的行动计划相对保守，工作生活之余把英语学习放在第一位，其他的事项原则上有剩余精力了推进一些，没时间就不强求。
英语：2月4日70min，2月5日80min，2月6日125min，2月9日110min，七天假期中有三天没有学英语。
阅读：重读《魔法岁月》前3章，做笔记，用时7h。
社交：2月4日（年三十）晚上到同事家串门闲聊90min，2月7日值班期间与读书相关话题的聊天约2h。
家庭：带鱼去鳞，包饺子，看电影《小猪佩奇过大年》，体验大别山温泉，陪媳妇看了央视重播的几个春晚节目，跟儿子下了几盘国际象棋，晚上睡前为孩子们讲故事。
微信：17年春节前夕叔叔建立了家庭微信群“耕读传家”，春节时无疑是群里一年最热闹的时候，除了红包炸群之外，你一言我一语争先恐后的语音消息让人体会到来自家人最真切的关心与祝福。
最后附上我年三十时发在家庭微信群里的祝福，与大家共勉。

2019春节寄语：

今年因工作原因不能回家过年了，在湖北遥祝咱大家庭所有成员新春愉快，身体健康！佳节之际多与家人亲友交流沟通，聊聊一年来的收获，谈谈下一年的打算。
愿大家在新的一年里学习上融会贯通，取得更好的成绩；工作中勤练技能，赚得更高的薪资；带娃时以身作则，为孩子树立良好的榜样；生活里乐观向上，为自己的目标持续努力。
社会中的竞争是十分残酷的，家庭是一个人成长的基点，也是每个人最强大的后盾，新的一年里，让我们互帮互助，携手并进，为更美好的明天奋斗不止！
