昨晚读了刘慈欣的一个短篇小说《诗云》，回忆起读研究生期间和舍友激烈争论“计算机能否战胜人”的场景。往事如烟，曾经的“赌局”亦随风而过。
##技术是反诗意的
小说《诗云》讲的是一个可以穿越十一维空间的神族挑战人类唐诗的故事。
在神族心中，他们的技术是可以超越一切的。可是当神汲取并融合了整个汉语的数据库以及关于地球历史的所有知识后，他沉默了。接下来他进一步利用基因技术把自己映射成为人类李白的样子，用纯能制造出了李白写诗用的书案和笔墨纸砚，甚至造出了烈酒并痛饮至烂醉如泥，但还是没能写出一首诗来。随后的两个月中神周游了整个地球，浪迹山水之间，饱览美景之后，依然没有作出像样的唐诗。
继而，神换了另外一种思路——暴力穷举，用量子计算机试便所有汉字的所有组合，写出所有的诗词；然后使用基于量子多态叠加原理的量子存储器结合传统存储技术永久保存这些诗作，终结诗词艺术。
故事中的人类则坚信唐诗是一种技术力量无法超越的艺术。
其中涉及到数量级的计算时采用了围棋来举例：粗略来说棋盘的状态数是3的361次幂，即10的172次幂，可是整个宇宙中的全部原子数只有10的80次幂个。利用神所掌握的最先进的技术来储存穷举出来的唐诗的话，共需要10的57次幂个原子……几乎是整个太阳系的物质量。
故事临近结束时，作为中介和旁观者的恐龙代表大牙仰头对着诗云大声问：“智慧生命的精华和本质，真的是技术所无法触及的吗？”
看完小说，准备睡觉，读研期间与宿舍同学争论的场景在脑海中涌现出来，久久不能隐去。
##对赌：机器人能战胜乒乓球国手吗？
争论源于舍友推荐我们观看的一个四翼无人机视频，视频中无人机可以自主追踪并触碰到测试人员抛向空中的乒乓球。视频观看完毕后舍友提出一个设想“用无人机跟人打乒乓球，无人机采用把每个球都打回去的策略，就可以赢得比赛”，然后我俩就该问题激烈争论了将近一个小时………

一开始，双方都凭感觉来设想问题的复杂性，主要是算法复杂度，就是说机器人从“对方击球后”开始判断球离开对方球拍的初始速度、旋转以及位置，计算球的飞行轨迹，到做出还击（当然要确保是合法击球，把打到对方球台）所需要的时间，当然还会涉及球从本方球台弹起后，选择在空中哪个点击球，怎样击球的问题。

双方的共同前提是随着科技的发展可以实现电脑打赢人，争论的焦点归结于在未来什么时候可以实现的问题上。也就是说机器人什么时候上述整个过程会比人做出相应的判断与击球更快、更准、更狠。我的观点是至少需要五十年，室友说绝对不需要那么久。争了几句后，我说要不咱们打个赌，五十年后看输赢。不料舍友却说：“50年后我认不认识你还不一定呢，怎么赌？”
这话听得我很不爽，当时就插入一个小支线的争论。
我说“你说这话就不爱听了，咱们一个宿舍的舍友，这样说多伤感情啊！“
“我没觉得有什么不合适。”
“要是50年后，咱们要是都能彼此记得，那时多么美好的感觉呀，你咋能这样说？”
插曲之后，话题继续…

接下来我抛出自己的判断：“网球项目比乒乓球项目简单些，网球项目要想让机器人在乒乓球项目上打赢人，必须要先在网球项目上打赢人。”双方在这一点上也达成共识。可我依旧坚持50年内机器人打不赢国家乒乓球高手。
舍友争论说：“你觉得不能短时间内实现的关键点在于哪里？”
“乒乓球的速度和旋转，以及对方发不出台的短球时机器人怎么回击。”
舍友拿导弹拦截和子弹飞行速度来反驳，我一时想不出来说什么，胡乱说：“你知道乒乓球有多重吗？怎么能拿导弹和子弹来类比？”（言下之意想说的是乒乓球的旋转会涉及到复杂的气体力学等）
他反问：“那你知道乒乓球多重吗？你也不知道吧？”
“2.8克左右。”
争论的过程中他在网上开始查乒乓球的重量，结果查出来是2.7克，他承认说：“这一点你是对的。”然后双方继续。
我拿人的进化来指出人脑及人体的精细控制和复杂性佐证，他用摩尔定律来反驳。他开始以18个月为周期来算50年内计算机性能的提升，我反驳说目前CPU工艺制成是20nm摩尔定律可能即将失效，他检索后结果显示可能在2020年后，每3年性能翻一翻，然后，算出2的16次方。来代表50年后计算机的性能是现在的倍数。并且引用霍金的一本书来对比人的进化速度，和计算机性能的提升。
我指出用电脑解决问题所需要建立模型的复杂性，以及时间复杂度。现在这个模型的复杂度很难衡量，继而两个人把话题引申到国际象棋领域，继深蓝战胜卡斯帕罗夫后，N多国象高手败给电脑已是不争事实。
“中国象棋领域人类也即将被电脑战胜，但围棋中电脑战胜人类高手还需要相当长的时间。”
“中国象棋领域人已经输给电脑了！”舍友立即予以反驳。
于是两人开始各自从网络上查询相关的资料，他找到的是 中国象棋关于人机对战的一个页面，我找的是知乎上的相关问题的回答。当时查询的结果表明，中国象棋方面许银川对战电脑保持和棋记录；普通的围棋软件只相当于业余4段水平。

我引用国象中目前电脑穷尽了10个子以下盘面的各种情况，以及策梅洛定理（Zermelo’s theorem）——在二人的有限游戏中，如果双方皆拥有完全的资讯，并且运气因素并不牵涉在游戏中，那先行或后行者当一必有一方有必胜/必不败的策略。（这里我的思路和《诗云》中神暴力穷举唐诗的思路如出一辙，我觉得只有得到一个计算好并储存下来的状态表，可以用于随时查询任意状态下的最佳策略，这样才算破解了一个游戏。）
他听后也立即说，这是用空间复杂性（存储）来换取时间复杂性（节省时间）是最笨的算法，实际上不需要那么做。
我坚持认为“实际的比赛中规则都是要限时间的，不可能让电脑无限极算，而职业棋手则有着较为灵敏的知觉和丰富的经验。”
两个人争得也累了，开始互相圆场，彼此找台阶下。

工作以来参加过三位同事的婚礼，都在我结婚之前。今天是第四次，也是婚后第一次。
##叙事
10:40分到酒店时，迎宾处和宴会厅的人寥寥无几，两个摄影师正忙着拍摄制作婚礼纪念影像的素材。随着新郎新娘的亲朋好友们陆续到来，迎宾厅及宴会厅瞬间热闹了起来，有的聊婚礼现场的布置，有的聊家乡的婚礼习俗。
“迎宾台布景上面酒店的广告看起来似乎不太和谐”
“应该把广告拆下来或者把布景布置的高一些挡住那些广告”
“车队应该是从……出发，然后……”
新郎新娘到达酒店后，大家纷纷至宴会厅就座，婚礼正式开始。是典型的套路：司仪上场、新郎上场、新娘在父亲的陪伴下出场、新郎迎接新娘、伴娘送上结婚戒指、新郎新娘互戴戒指、双方父母上场、新郎新娘敬改口茶、两位父亲致辞、新郎致辞、新娘抛手捧花、礼成开宴。
##感悟
有多少人还记得自己在婚礼上的承诺和誓言呢？新郎一句“只要你不离，我一定不弃！”不禁使我想起了近日生活中的点点滴滴，有些后悔没让妻子前来参加。恰如面向党旗重温入党誓词能使人铭记入党的初衷，参加婚礼能让人回忆当初走进婚姻的决心以及曾经许下的诺言。婚礼现场不仅是新郎新娘的幸福时刻，同样是已婚夫妻建构亲密关系的绝佳情境。
金庸借《鸳鸯刀》中林玉龙之口说：“不吵架的夫妻便不是真夫妻。”这话大体上是不错的，来自不同家庭的两个人，在诸多方面都会持有不同的观点和态度，行为模式也必然有着不小的差异，要想凡事都达成默契不争吵，通常来说需要相当长的磨合期，直至两个人都真正成熟起来——更深入地认识自身以及对方的信念、判断、态度、言语习惯和行为模式，爱伴侣和子女胜过爱自己。
记得我和妻子认识之前，大表哥曾说过“不要在意人样儿长哩好不好，要找个能过日子才中。”什么是过日子？说开来无非是彼此陪伴，相互理解支持，共同为家庭的幸福而努力奋斗，尽力使家中的物质条件一年比一年好，精神生活一年比一年丰富。
##书摘
《社会心理学》社会关系篇“吸引与亲密：喜欢他人与爱他人”一章，作者在后记“经营爱情”中写道：现代生活的两个事实似乎无可辩驳：其一，亲密而持久的婚姻关系是幸福生活的标志。其二，亲密而持久的婚姻关系正在减少。
关于如何付出努力以防止爱情的衰退，作者给出了如下的建议：

  • 每天挤出时间来聊聊当天发生的事情；
  • 克制自己的唠叨，不争吵，袒露自己并倾听对方的感伤、关注和梦想；
  • 努力使婚姻关系达到理想的完美境界，伴侣双方都能够自由地给予和获取，能够共同作出决策并一起享受生活。

愿天下有情人终成眷属，努力使婚姻关系达到理想的完美境界！

追寻写作路上的足迹，几度雄心壮志，几度无疾而终，致使自己的认知、思考水平至今仍在低层次徘徊，无实质性进步。分析起来，共有三个经历使得我对写作念念不忘的，现分述如下。
###1、《师友杂忆》
“既到老不忘，则可确证其为余生命中之重要部分，务求叙述真实，亦属余对生命之自惜。”钱穆在80岁高龄，历时5年著述成书。
2010年特有的心境下，拜读大师《师友杂忆》一书，感触颇多，头脑中首次萌生系统地回顾自己成长历程的想法。八年眨眼一瞬而过，试想50年后，我会如何回顾这一生呢？
###2、《最后一课》
2013年初，观看李开复推荐的演讲“兰迪 波许教授的最后一课”，在讲座结尾处兰迪揭示了至关重要的两点。

  • “关键不在于如何达成梦想，而是如何无悔度过人生，如果你正确引领你的一生，因缘自有报应，梦想自会实现。（It’s not about how to achieve your dreams. It’s about how to lead your life. If you lead your life the right way, the karma will take care of itself. The dreams will come to you. )
  • “这场演讲不是为你们准备的，而是留给我孩子们的叮嘱。”（The talk’s not for you guys, it’s for my kids. )

2017年12月重温这一演讲时，糖豆已近三周岁，思考自己当下的生活，脑海中不止一次地设想如下场景:儿女在今后成长过程中遇到困惑，问:“爸，您像我这么大的时候是什么样子，都在做什么？”我该作何回答。
###3、《奇特的一生》
”看来应当由每个人自己来制定自己的时间使用法，但不管怎样都应当深思一下自己的生活在怎样度过。“
大四时曾用博客粗略记录自己的生活，工作后用Excel记了3年时间账(2010-2012)，2016年3月研二下学期进入课题研究的关键阶段，阅读书籍《奇特的一生》后，开始实践其中的时间记录法，持续记录至今。
##行动计划
持续写作可以达到三个目的:1.提升认知能力；2.增强生命体验；3.明晰前进方向。通过写作将生命各阶段的感悟体会存盘:
( Present ):生活感悟
( Learn )：知识积累
( Read )：阅读心得
( Tools )：工具应用
( Game )：休闲娱乐
( Memory )：成长经历

如何离线安装python包

b、下载安装对应版本的.whl压缩包
前往pip官方源https://pypi.org/或国内pip源 https://pypi.tuna.tsinghua.edu.cn/simple/ 进行手动下载对应包的.whl文件，这里推荐使用国内pip源（后者）进行下载，下载速度较快。
由于whl是深度学习依赖包对应的压缩包，在其中包含了py文件，以及经过编译的pyd文件，因此可以直接进行安装，安装命令为：

pip install *.whl

c、下载安装.tar.gz或.zip源码包
在pypi的官网或者国内源pypi包中有些并不包含whl压缩包，但一般会有.tar.gz或者.zip的源码包，对于下载的源码包需要通过解压后得到的setup.py进行编译再安装，编译的命令为：

python setup.py build

安装的命令为：

python setup.py install

Reading

读钱穆《中国思想史》和《人生十论》

[《诏问山中何所有赋诗以答》]
[问我何所有，山中唯白云。]
[只堪自怡悦，不堪持赠君。]
[陶弘景]]

自11月6日隔离生活起，开启了新一段的读书旅程。原以是为隔离三天，只准备了《中国近代史新编》和《浮生六记》两本十月份没读完的书，进度分别为50%和35%；实际上隔离了八天，读完《中国近代史新编》后，利用相对集中的自由时间读完《认知心理学》，又开启了《一本书读懂台湾史》的阅读。结束隔离时，单位年终检查已经开始了，这意味着工作进入相对轻松的阶段。

隔离之后，连续两个周末都处于封闭管理的静默状态，19日早晨醒得比较早，05:00时洗漱好开始阅读《中国思想史》一书，接下来连续五天内用时13小时读完。正如2019年读《中国通史》的感觉，春秋时代的百家争鸣是中国思想史上最璀璨的篇章。全书读下来，除了春秋战国时代之外，印象最深的要属书目收尾处讲解“孙中山”的一节。

严格来讲，近五十年来，中国亦并无所谓思想界。只有孙中山一人，他终身从事革命的实际工作，固不该专以思想家目之。但中山先生实有他独特的一套思想，他不仅堪当这一百年来近代中国唯一的一个思想家，而且无疑地他仍将是此后中国思想新生首先一个领导人。
马克思以物质为历史的重心是不对的。社会问题才是历史的重心。
人类求生存，才是社会进化的原因。阶级斗争，不是社会进化的原因，而是社会当进化时发生的一种病症。
孙中山的《三民主义》，应该是近代中国新生唯一的启示。若我们把一个人的生命作比喻，民族主义是其人之精神与灵魂，民权主义是骨骼，民生主义是血肉。三者缺一不可。

读完《中国思想史》后，又相继在三天内用时9个小时阅读了钱穆大师的《人生十论》，这本书读来是自己的内心趋于平和，与今年七月至十月间的急躁形成鲜明对比。
钱穆认为，人生是“事业”与“性情”的一体两面，奉劝大家，“莫要太看重了外面事业，而忽略了内部性情。”

性情不是天生的，需注意后天培养。从个人言，各该当心不断自修自养。从大群言，我们要认识中国人性情来培养我们自己的性情，从中国的文学与艺术中多所采撷，来帮助自己作人生的修养。
