##L5
a 和 an
Sophie is a new student. 索菲娅是一名新学生。
This is an umbrella. 这是一把雨伞。
这两个句子中出现的a／an在英语中被称为不定冠词。a和an在意义上没有区别。用a／an时，我们必须记住两条基本原则：
（1）a／an有不确定的意义（即所说的人、动物或东西对听者或读者来说可能是不知道的）。
（2）a／an只能用于单数可数名词之前。
此外还需注意a和an的发音：a（在平时讲话中发／+／音）用于辅音（不是辅音字母）之前；an（／+(／）用于元音（不仅仅是以元音字母a，e，i，o，u开头的词）之前。当我们把a或an用来说明字母表中的字母时，便会一目了然：
This is a B／C／D／G／J／K／P／Q／R／T／U／V／W／Y／Z．
This is an A／E／F／H／I／L／M／N／O／S／X．

成为建设者，而不是堕落者

不知不觉间，时光悄悄溜走，今日已是三月的最后一天。下午带孩子们户外活动近一个小时，回到家后想诱导他们学国象时，被俩人婉言拒绝。我茫然若失，瞬间连下一局棋的精神都提不起来！《月亮与六便士》的剧情再次涌入脑海，主人公无疑是幸福的，无论生活上多么狼狈拮据，他还有自己心灵的港湾——画画。我呢？假期里除辅导糖豆俩学了一些汉字和字母外，时间几乎全都虚度了，消失得无影无踪。

这个特殊的假期整体上分为三个阶段：
##走亲访友
1月20日驱车回到伊川，计划先去县城的丈母娘家待几天，然后再回乡下父母家里。不巧岳父岳母他们没在家，我们索性去了姐姐家。姐姐和姐夫俩人出去忙事情还没回来，只有两个外甥带着小外甥女在家里。坐下来同他们聊天，才知道原来连午饭是大外甥做的，午饭后，两个人轮流带妹妹，都已经到小区下面理过发了。问起大外甥《哈利波特》有没有看完，他回答说：“已经看过两遍了！”说完，他又拿出最近刚看完的几本书，饶有兴致地为我们讲解书中的故事。晚上到岳母家，和小舅子亮亮、内表弟东东一起在南街菜市口吃刀削面、烤鱼。聊天中东东说起自己关于买房的思考，让我有些惊讶：虽说东东毕业后在县城工商银行上班还不到半年，但不得不承认，他的想法比我大学毕业那会儿成熟得多。（或许这就是爱情的力量吧^_^）
第二天早饭后到我叔家串门，叔叔和婶婶都没在家，家里只有小堂弟一个人。两年不见，堂弟个头已经超过我啦！接下来去大堂兄家，和堂兄堂嫂她们聊得很开心。
第三天午饭后开车回家，到家门口时发现爸妈早已打开了铁大门，车直接开到大门楼下面停好。（乡村中流行安装两扇对开的铁大门，其中一扇门上再开一小门供日常走行人，过农用车或必要时才会将两扇大门都打开。谁会想到呢？车在大门楼下一停就是40多天，准备开车时才发现车的右前轮已经没气了！）
第四天（1月23日，农历腊月二十九）：上午到二伯家和大伯家串门后，和爸爸一起贴春联。
1月24日（腊月三十）：上午到杨庄接弟弟回来；晚上和家人一起看春晚，有几个节目还是很棒的。因为疫情的关系弟媳不用大年初一值班了，晚上也回到我们家。
1月25日（正月初一）：假期第一天早起，爸爸烧香，我生火烧好了家人洗漱用的热水后，写了这篇春节记事的前面几段文字。早饭后带妻子、儿子女儿前往大伯二伯家拜年，听伯父和堂兄他们聊家常。午饭后组织家人打够级，还没打够一圈呢，村里有个叔叔来找爸爸聊天，牌局散了。临近晚饭时，分别和儿子、女儿下了一局国际象棋，晚饭后爸、妈、妻子和我四个人打升级。临睡前，儿子女儿再次开启十万个为什么模式，我困得上下眼皮儿直打架，回答了他们长达四十分钟的各种提问。
1月26日（农历正月初二）：按习俗出门的女儿是应该回娘家的，今年因为新冠状病毒引发的肺炎疫情的原因，几乎没有人走亲戚。下午教家人们打保皇，妈妈和爸爸在判断敌友上有些迟疑，导致我们几个也容易被迷惑，每一局都十分有趣，整个下午屋子里充满了欢乐的笑声。

##疯狂娱乐
自1月26日开始带着家人一起打保皇，随着假期因疫情不断延长，我们的娱乐也无节制地升级——接下来近一个月的时间里，纸牌玩腻了就换麻将调节一下，平均每天用于打牌的时间长达四个小时。。。2月22日县城解除交通管制，弟媳妇接到明天上班复工的通知。恰好同村有人开车去县城，弟弟陪她搭车一起前往。他们去年结婚时，我正在重庆参加一个培训，没能赶回家参加他们的婚礼。今年我带着妻子儿女回家过年，因为新冠状病毒肺炎疫情的原因，我们一起在乡下家里共同生活了一个月，对彼此也有了更多了解。
弟弟和弟媳去县城后，不知道下次家人们聚在一起打保皇是何年何月？！
这段时间中，打牌之余我刷完了电视剧《平凡的世界》，在chess.com上疯狂下棋，快棋战绩最高时达1190分。

##尝试向建设者回归

2月14日，叔叔在家庭微信群中转发了一篇题为“居家隔离21天后，一个普通人的15条教训”的文章，我仔细看完后觉得文章写得非常棒，尤其是第十四条成为建设者而不是堕落者的一段文字。

居家隔离的日子，也是闭关修行的日子。如果我们在这段至暗岁月里，能够整合自己的生活态度，重塑内心信念，审视余生之路。等艰难的时刻过尽，我们将拥有更深沉与更健康的自我。灾难之时，也是修行之日。不要堕落，不要在无序中无爱，在无聊中无德。当一个建设者，承担起自我的责任，在心中生起一团火，让更多人借光走出黑暗。

2月16日，我开始陆续教孩子们学习写一些简单的汉字，20以内的加减法，几天后又加上了字母和国象的教学。
23日刷完《平凡的世界》，疯狂娱乐模式总算告一段落，我以“成为建设者而不是堕落者”为题写了一篇文章，推送至微信公众号，文中简单反思了自己的假期生活。接下来，我整理以前写的半成品文章，连续在公众号推送了五篇文章。
第六天（2月28日），再次坐下来想写一些东西的时候，我卡壳了，思绪开始烦乱，无法静下心来。我邀请照凯哥杀了两局围棋。复盘时照凯哥总结说“中腹不好占……你再练上一年应该就会比我下得好了！”其实我也知道“金角银边草肚皮”，只是下快棋时往往来不及仔细计算，结果走着走着我就围起中腹了。我觉得自己的围棋水平和表格旗鼓相当，虽说两盘我都输了，但盘面差距都在十目以内，而我围的是中腹啊！所以我有些不服气，想赢回来。3月1日，我在腾讯平台上下了四五局棋，想找回一些棋感。结果发现自己依然很难适应快棋的节奏，同时觉得腾讯的界面不很友好（平台保存对局棋谱方面也不够人性化：如果没有及时设置保存的选项或者手动保存，棋谱就丢失了；弈城客户端会自动保存每一局棋谱到服务器，随时都可以下载）。3月4日，我重新登录弈城客户端，查看对局记录发现最后一次对局记录是2019年4月11日，将近一年没下围棋了。接下来十二天的时间里，我一共下了十八盘棋，平均每天1.5局。只赢了其中的七盘，其它的都输了，所幸的是保住了5K没有被打降级。3月15日小妞要回娘家，我也跟着一起去带孩子，“找棋感”的进程暂告一段落。

##岳母家
3月28日自县城返回乡下，我们在岳母家共计待了十四天，这段时间里时间里，陪娃写字儿、玩耍，帮忙做少量家务之余，国象再次成为我生活的主旋律。13日购买chess.com会员后，APP内一直没有解锁相应的功能。写邮件想客服求助，用蹩脚的英文数次沟通后，我终于获得了会员资格（我最看重的是复盘棋局时的电脑分析功能）。有一天晚上，我输了再战，再输，再战，chess.com上五分钟快棋的分数直接掉到了1000分以下！（二月份时用了将近一个星期的时间，才好不容易升到近1200分。）此外还关注了网络亲子赛的决赛，以及候选人赛事。
其他值得一提的，有下面几个事儿：

  • 听岳母唠家常，讲述她的成长经历；
  • 用跑步机跑步三次，一次两公里，两次三公里。
  • 检测小舅子回收的sony笔记本电脑性能；
  • 整理优化岳母家台式电脑；
  • 将岳父旧手机照片转存至新手机；
  • 维修故障计算器。

##即将离开家的几天
3月28日孩儿他妗子送我们回来的当天，我们带着孩子们去野外沟里面玩了一阵子。接下来的几天，天气降温，还时不时夹杂着小雨，让人心情也闷闷的，高兴不起来。
3月29日04:30就醒了，再无睡意，听了凤凰媒体就《任志强自传》对任志强和潘石屹的采访录音。白天重温《大话西游I-月光宝盒》
3月30日，想在网上采购一批书籍，挑来挑去加了几本书到购物车，最终没下定决心购买——武汉市将于4月8日起解除离汉离鄂通道管控措施，我复工的日子应该也不远了。
3月31日：整理电脑中文件，起草这篇文章
4月1日：接到领导协调部门人员复工的电话。
4月2日：前往县城，向岳父母道别，同岳母一起去外婆家。
4月3日：收拾行李，整装待发。给爸爸过五十八岁生日，虽说爸爸嫌蛋糕性价比低，可是当孩子们唱响生日快乐歌，看到爸爸脸上徜徉着幸福的笑容，我们觉得这个蛋糕无疑是超值的。
4月4日傍晚时确定了次日出发，在小妞的坚持下我们去超市进行大采购；然后到我三姑家去了一趟，听她唠唠家常。回家后行李装车，爸妈督促着带足隔离期蔬菜和粮食。

##自我隔离的日子
4月5日早上06:40起床时，妈妈已做好早饭，煮了鸡蛋让我路上带；爸爸去邻近的村子买了一大袋馒头回来，在门前的菜地上刨了一些大葱，妻子无言地挖着菠菜……饭后07:40出发，家人们再三叮嘱我路上开车千万要注意安全，每1~2小时休息一次。……不得不承认，长时间的开车确实让人很容易疲劳，尤其是最后的两个小时。17:30到达黄冈家里，感觉整个人都累爆了！
吃过晚饭稍事休息，把携带的行李一一拿上楼，摆得客厅简直无法下脚。简单整理后，开始筹划隔离期间的生活。

  • 计划每天阅读时间不少于3小时。选了六本放在案头，分别是《进化心理学》、《封神演义》、《后西游记》、《阅微草堂笔记》、《小王子》和《你是人间的四月天》。
  • 网上下单采购鼠标垫。假期里面用媳妇的笔记本电脑，感觉到诸多不便。这下到了家里，台式机性能胜过笔记本百倍，自己安装的软件用来也十分顺手，应该是可以大有作为的。比如多推送几篇公众号文章、在知乎上回答一些问题、尝试在B站发个原创视频等，尝试打造一下个人IP；或者写个Python程序，实现爬取chess.com对局信息的功能（假期中已有构思，安装Pycharm后中止）。
  • 购买《少儿国际象棋-中级篇》。参考林峰老师的《国际象棋初级教程》系统学习大师三人行系列教材，适当实战并及时复盘，提升自身国象水平，为教孩子们学国象打好基础。

#谁言寸草心，报得三春晖
2019年12月24日星期二

转眼间，爸爸远离家乡来到黄冈帮我们带糖豆已经快两年了。今年11月妻子参加完消防工程师考试后，爸爸果断地提出，要在接下来的一个周末回洛阳。我们似乎再也找不出能让他继续留在这边的理由。
妻子刚考完试的时候，爸爸就提出让我们赶紧购买车票。我一直拖着没买：车票没定，心理上就觉得爸爸还可以和我们在一起很多天；车票定了，一个星期仿佛眨眼间就会溜走。
11月15日星期五，我正常过周末，刚到家，爸爸就再一次地督促着让我赶紧买火车票。我提议让他坐周日下午从武汉出发的高铁，这样的话早上早些出发送他到武汉，利用上午的时间到黄鹤楼看看。爸爸没同意去黄鹤楼的提议，坚持让我买上午出发的车次，以便下午到达洛阳后有充足的时间，好往乡下的家里赶。
晚饭后，妻子带孩子们先到小区院子里去玩了。爸爸像往常一样准备下楼倒垃圾，临出门前爸爸淡淡地问：“林家大湾是不是离市区不远？”
“嗯，离市区不远。”
“那咱明天去那儿看看吧？”
“好。”
爸爸下楼后，我边刷碗边纳闷儿，爸爸拒绝了去黄鹤楼的提议，咋会突然想起来要去邻家大湾了呢？
妻子和孩子们回来后，我说明天计划带咱爸去林家大湾一趟，妻子欣然同意的同时脸上露出诡异的笑，猜测说：“这一定是咱爸的主意吧！”

16日到了林家大湾，第一个景点军事展览馆，还没看多大一会儿，糖豆俩就没了兴致，我只好带着他们爬上阁楼，看看周围的山景。从阁楼上下来，豆豆对展厅里的几幅地图比较好奇，我带着她俩找展厅里张挂的地图。我们看完地图后就从展览馆里面出来时，妻子早已在门口等着了。妻子在周边买了炸薯片哄糖豆，我们在门口等着爸爸。爸爸认真仔细地看了展厅中的每一幅人物介绍，展柜里的每一样物品。
接下来的景点是名人故居，包括帅门、林育英故居、林彪故居等。一开始糖豆指着故居中的一些纺花车、织布机、磨盘等物品问东问西，有些物品我也没见过，让他们去问爷爷（这些旧物件几乎都是爸爸用过的，至少是见过的）。爸爸饶有兴致地介绍了几个，正在劲儿头上呢，糖豆俩却又开始像钻迷宫一样多起了躲猫猫，我和妻子生怕他们迷路或者跌倒，也只好跟着跑前跑后。没多大工夫，糖豆就带着我们快速浏览了接下来的景点。
糖豆俩有些累了，我们回到车上吃带来的零食和水果。爸爸一个人在后面逐个屋子地观看名人故居，每一个人物介绍都看的相当细致。
游览结束时，我和妻子一直认为，爸爸帮我们值回了票价。回市区的路上，糖豆俩睡着了，与爸爸闲谈的过程中才明白——爸爸之所以想到参观林家大湾，源于我的一个表叔曾问过他“在黄冈一年多了，有没有去林彪故居看看？”
11月17日早饭后送爸爸至黄冈西城铁站，回到家中看到衣柜角落里整齐地摆放着两盒纯牛奶、三个苹果……。那些东西原本是前一天晚上装在爸爸的包里，让他带在路上吃的，他把糖豆喜欢吃的都悄悄地放在柜子里，只带了面包和一个苹果。一种说不出的滋味儿涌上心头，瞬间又转化为无奈。回想我求学期间，每次离家出发的时候，爸妈总是想把家里好用的、好吃的塞买我的行囊。我呢，喜欢的就带一些，不喜欢的一点儿也不愿意往包里装。有一次，妈妈煮了鸡蛋让我路上带着吃，我一个也没带。后来听爸爸说起来，我走后，妈妈伤心了好长时间……不觉间，泪水已模糊了视线。

12月4日爸爸在家庭微信群中发了一条消息：
“我11月17日回到了家，18号安排了家里生活，19~20号帮亲人务工，20日晚上来到了凯地城，我的确过得很充实。我也遗憾没挣很多钱，我也高兴没伤害任何人。”

我很清楚爸爸想要表达的意思，他想说尽自己所能照顾到了我们所有人。爸爸何曾伤害过我们中的任何一个人呢?他倾自己所有的金钱和时间，力所能及地帮助我们姊妹兄弟四人，盼我们能过上好日子。
