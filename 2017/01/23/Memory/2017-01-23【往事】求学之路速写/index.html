求学之路速写

【往事】系列用于简述我的成长经历，聊聊记忆中的往事。

拥抱命运一切都是最好的安排

人择原理：“我们看到的宇宙之所以是这个样子，乃是因为我们的存在。”
[——《时间简史》]]

##1
丁卯年己酉月甲戌日，LY与ZZ地界交汇处小村落的一户人家里，一对儿夫妇为了能尽早把采摘到家的烟叶装进炕里烘烤，持续干活到深夜。忙完之后，女子淡淡地对男人说：“你赶紧去找接生婆吧，我觉得有阵儿了，可能天不明就要生。” （//：其时女子已有九个月的身孕，一天的劳累对她来说意味着什么，今天的我们已经无法想象。）
黎明时分，该女子诞下一男婴，母子平安；我健康、幸运地来到这个世界。
##2
五年后的一天，爸爸抱着我去XB小学，我哭了一路。到学校后，爸爸把我放下来，让其他小朋友在校园里喊“老师来了，老师来了……”小朋友们的呼喊声中，爸爸把我留在那儿，离开了学校。 （//：当时我被吓得躲到教室里的桌子下面，而不是往学校外面跑，今天想来也是佩服我自己；同校的还有我的姐姐和一个堂姐，入学前在爸爸的谆谆教导下我已经会写100以内的阿拉伯数字。）
二年级暑假，王老师调离XB小学；爸爸将我转至XX小学。 （//：当时XX小学三年级班上已没有更多的空位，爸爸将家里的桌子精心改造之后扛到学校给我当书桌用；这桌子上应该还有我刻的“早”字儿。）

##3
小升初时面临学校的选择时，爸爸毫不犹豫地将我送至一中。 （//：一中在镇上离家8km远、教学质量时镇上是最好的；三中在临近的村子里离家4.3km、教学质量一般；当时堂兄恰好在镇上一中教书。）
初一暑假临近开学时，我曾对父亲说：”我不想去一中读书了，镇上的不良少年总是欺负村子里去的学生！“第二天大堂兄来家里跟我说：”学校的校长换了，将会大力整顿学校风气，再开学就会好很多“。我沉默许久，开学后继续到一中念书。 （//：小学毕业班42名同学中只十个左右到一中念书，初二开学时半数转到了三中）

##4
初升高成绩公布后，我差6分没过县第一高中的录取分数线。如果去一中念书需多缴纳￥2900借读费。当我说自己不想复读时，得到的是爸爸干脆利索的回答“不复读就不复读，咱上一中”。 （//：初中的好多同学选择了县里的第二高中，也有大部分人选择了回读初三；高一入校时我全年级排名第480；高二时我最好年级排名第9；高三包括复读班在内，年级最好排名第21。）

##5
2004年高考后6月9日自己估分613，13日到学校报志愿。好不容易从班主任老师那里争取到一张零表，华丽地填写了国防科大和复旦大学。 （//：由于是估分填志愿，在HN这个人口大省，大多数人填零表其实只是给自己多一线录取机会而已，零表大家都会使劲儿朝高了报，当时还是手写涂卡二合一那种传统的纸质志愿填报表。） 一表经全家人再三斟酌后三个批次分别报了华中科技、郑州大学、太原理工。
7月下旬，当大堂姐从网上查询到我的录取结果为HGC时，所有希望瞬间化为泡影，我陷入了深深的沉默。接下来的几天内，爸妈曾多次询问我的打算，我嘴上说等拿到通知书再做打算，心中实则举棋不定！ （//：再战高三吗？可是自己真的已经足够努力了，623分全班第二是我历来最好的成绩；英语和语文是自己的硬伤，多读一年究竟会有多大提高是个未知数；读HGC这个从未有所耳闻的大学，又觉得有些许不甘……）
8月3日去学校取通知书的路上，我心中依旧没有答案。但到学校后的见闻，让我在直觉上得出有半数以上的同学选择复读这一结论，心中第六感暗暗地告诉自己“不能复读”；拿到通知书去见班主任时，赵老师说“你这是定向生，国家每年还给发￥5000的奖学金。”我心中疑惑“是这样的吗？”
十天后我在父亲的陪同下踏上大学征程。
