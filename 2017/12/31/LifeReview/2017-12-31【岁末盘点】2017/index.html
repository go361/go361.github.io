生活速写

岁末盘点2017

生命如白驹过隙，转眼又到岁末总结时。今年有3月1日“2017年度计划”所立的各种Flag为参照 (虚线方框内为计划) ，自己那可怜的行动力、执行力只能是理屈词穷，无言以对!

Flag1: 把家庭放在首位

目标：与家庭成员和睦相处，不宣泄负面情绪
行动方案：1.与家人相处时，把个人的事情先放一放；2.每天至少与家庭成员沟通 30min；3.重读《高效能家庭的7个习惯》，在生活中践行书中好的观念方法。

[年度行动成果]

  • 除9.28~10.13日相聚期间，对自己的表现还算满意外，在同家人打电话沟通中鲜有全神贯注的情形。
  • 5月1日开始工作至今:与家人累计通话时长70h，平均下来每月9个小时、每天18分钟。其中与妻子通话占90%以上，与父母、兄弟姐妹沟通甚少!
  • 压根儿没翻开《高效能》一书……

Flag2: 培养情绪控制力

目标：与人相处过程中无过激言行
行动方案：将情绪控制这一项纳入每日复盘内容。认真记录情绪波动情况，分析自己做得好和不好的地方。
时间安排：每日 20~30 分钟。

[年度行动成果]

  • 工作中言行明显欠妥情形5次
  • 每日情绪复盘行动量 0

Flag3: 调整身体锻炼方式

目标：5公里成绩25min以内；羽毛球步法合理化；在健身方面探索并做出尝试；
行动方案：1.每周跑步2次（每次30分钟）；2.每周3次羽毛球步法练习；3.找到健身的小伙伴并参与锻炼活动；4.学习跑步及健身相关知识《身体锻炼手册：真实年龄健身计划》，陈伯龄的酱油台 公众号文章等。
时间安排：平均每日1小时。

[年度执行成果]

  • 跑3km累计仅 7次(4.5h)，最佳成绩15’07”
  • 健身方面累计投入时长 3h
  • 羽毛球(127h):步法练习不足5次，同单位外球友交流5次，球技仍是原地踏步，未见些许提升

Flag4: 英语学习

目标：阅读英文原版文学作品
行动方案：1. 每日完成朗读任务；2. 牛津高阶词典APP中完成每日学习卡片（20个），从3000核心词汇开始；3. 通过手机听英语系列节目，增加自己暴露在英语环境中的时长。
时间安排：每日60分钟。

[年度执行成果]

  • 朗读(55h):最长无间断行动15天，最长中断时间持续15天，输出语音78min；
  • 主动单词复习(3.5h)
  • 音视频材料(3h)
  • 读完《half girlfriend 》(22h)

Flag5: 程序设计

目标：尝试写出数独的辅助解题程序
行动方案：1.学完《MFC教程》一书及视频；2.阅读《程序设计艺术》一书中基础理论部分内容。
时间安排：每日40分钟

[年度执行成果]
10月20日开始自学 Python，累计学习时长88h，目前进度:

  • 《父与子的编程之旅》约32小时
  • 《核心编程》11.19-12.5
  • “网易云课堂” 约17小时
    自写数独程序，只实现了最基本规则，仅能解决最简单级别的数学游戏。

Flag6: 阅读与写作

目标：践行输出倒逼输入的理念，每周撰写1 篇阅读心得
行动方案：1.养成每天读书的习惯（30min）；2.每周进行总结，撰写体会（800字），并结合自己的生活落实到行动上；3.在单位内部筹划读者俱乐部；
时间安排：每日30分钟阅读；每周2h写感悟。

[年度行动成果]

  • **阅读(350h):**《故事》《褚时健传》《认识商业》《人类简史》《社群主义》《国富论》《亲密关系》《未来简史》《刻意学习》《拆掉思维里的墙》《战争论》《三联生活周刊 2017-29期：封面故事(金庸)》《对抗语文、语文是什么》《知识大融通》《文章作法》《金融心理学》
  • 读书感悟撰写 0 篇，制作<战争论>序言和第一章的思维导图，摘录了<奇特的一生>部分内容。
  • 筹划俱乐部尚在酝酿中

Flag7:围棋

围棋自学进入瓶颈期，需重新夯实基础，更加系统地学习。
目标：弈城围棋平台升到业1级别
行动方案：1.继续学完《围棋一月通丛书》（开局、中盘、手筋3个分册）；2.订阅《围棋天地》杂志，购买《新世纪围棋之魅》（套装全4册）；3.阅读沈果孙系列著作；4.尝试在单位成立围棋兴趣小组，开展相关活动。
时间安排：每工作日30min学习；每周 6h 对弈实战（下完当即复盘）。

[年度行动成果]

  • 3月以来累计投入280h(平均每天1h+)，弈城级别由8K提升5K
  • 学完《围棋一月通》中盘册
  • 没订阅《围棋天地》；买了《新世纪围棋之魅》，但尚未开始学习
  • 《围棋正着与俗手》阅读至第2章后送人了，《围棋常型百例》46~100，《围棋常见死活题》学完第一遍
  • 目前尚未在单位内部找见棋友，共去东坡棋院2次主要是和围棋老师聊天，看别人下棋
  • 为观看AlphaGo与柯洁的对局做铺垫，阅读了《石破天惊》《围棋人机大战》《对面千里》《AlphaGo60连胜》，关注AlphaGo与柯洁的对局，后期观看了樊麾、柯洁在围棋TV复盘人机大战的节目。
  • 打谱完成《百年围棋经典名局》(9~30)、《大师对决》
  • 阅读《中的精神》、《无心》、《不得贪胜》3本名人自传
