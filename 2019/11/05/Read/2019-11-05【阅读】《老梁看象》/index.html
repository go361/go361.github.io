Reading

《老梁看象》

已经记不清之前是在什么时候、什么地方看过一期老梁的节目，里面讲述了象棋界前辈杨官璘的故事。

[梁宏达·著]
[社会科学文献出版社]
[2018年7月第一版]

读书情景

读书期间：2019/10/11~14日，
累计用时：155min

https://book.douban.com/subject/30423910/

十月十一日我带着《西游记》陪妻子去HG市图书馆，读完《西游记》浏览书架上体育类图书时，发现了《老梁看象》这本书。把这本书借回家，当天晚上一口气读了四章，昨天读了两章，今天午睡前读完了全书。书中第七章“象棋泰斗杨官璘”的内容正是我之前看过的那一期电视节目。

序言

序言中对这本书有如下介绍：

这本书的内容源自2013年中国象棋甲级联赛期间，老梁作为象棋文化推广大使，应中国棋院和中国象棋协会的邀请，为央视录制的《老梁看象》专题节目。以象棋为主题，从江湖象棋入手，通过轶事趣闻、民间故事、棋局讲解、残局分析等形式，生动讲解了江湖残局、盲棋、彩棋、让先与让子棋的来龙去脉，以及彭述圣、董文渊、杨官璘等老一代棋人的陈年往事。

内容简介

象棋无疑是国内最接地气的体育运动之一，老梁一开篇比较下象棋与打麻将的区别，说象棋这项体育运动的魅力在于“不动输赢（不下注赌钱），也能让三亿多人忘我地投入其中。”这话真是一语中的——天气好的时候，在城市里街边小巷中走走，准能遇到一群人扎堆儿围着棋摊儿，你一言我一语地为对局者支招——喜欢下象棋的朋友们对于这一场景应该是再熟悉不过了。
讲盲棋的一章，先说事实“象棋在我国当今社会的影响力呈下降的趋势”，然后引出最神秘、最有特色、最利于象棋推广的形式——盲棋。然后指出练盲棋需要具备的两个主观条件“动态记忆力好”和“熟记各种棋谱”，一个客观条件“对方起手的棋艺水平要低一些。”最后总结说，下盲棋虽然很能显摆才智、赚人眼球，但这显摆是拿功夫换来的。正所谓“台上十分钟，台下十年功。”
讲到让子棋时，老梁仔细讲解了让先及让子的规矩，比如棋子不能“过河”、不能吃子、每个棋子只能走一步、“铁车”、“铁兵”、“三做功”等，这些讲究，是为了让整盘棋子力均衡，达到更有效地进行对弈的目的。
讲彩棋（下赌注的棋）时，开宗明义地说，棋类项目有一个特点，一个棋手的棋力在通常情况下是比较稳定的，很少出现“超水平发挥”或受意外因素而大幅度改变实力的情况。总结道，彩棋是一定历史阶段的产物，这项活动无论对社会风气还是对人民生活而言都没有好处，随着社会的发展和进步，彩棋已经退出了历史舞台，其正面作用已经完全被其他象棋活动所取代。
第六章讲棋坛双枪将董文渊，“双枪将”是指一个棋手在棋坛领域中的一个棋种上，竞技水平已达到了极高的水平，同时在另一个棋种上也能达到专业棋力水平。这位难得的棋坛双枪，一生算计，处处以己为先，恃才放纵，最终不为人接纳，其晚年惨淡，令人唏嘘。作者以此为例，劝诫读者做人要厚道——“一个人有没有才华、有没有成就、能不能当红，可能与做人没什么关系，可是能红多长时间，路能走多远，就和做人有很大关系了。”
杨官璘是旧中国最后一个棋王，也是新中国的第一个棋王，在讲述他的故事时，老梁有如下总结

有道是“干一行爱一行”，所有的努力都是有回报的，年轻的棋手应该学习老杨沉下心思，潜心研究的态度和努力。

讲解“南杨北王”（南杨即是指杨官璘，北王指王嘉良）的一章中，老梁对两个人参加第一届全国个人赛的故事由这样的总结——“无论多高明的象棋选手都有短板，即使是棋技功夫都修炼到家，但是在性格、情绪、思维方式乃至生活习惯上的一些弱点，也会导致下棋的失误。”
最后一章落脚在象棋甲级联赛，叙述了比赛中使用电脑软件作弊的一些故事。自97年国际象棋项目中“深蓝”战胜卡斯帕罗夫以来，经过近二十年的发展，计算机软件的分析系统几乎可以涵盖象棋盘面上的所有变化，然后从中选出最佳招法；再加上象棋比赛有时间限制，电脑又是“铁石心肠”，没有人类的情绪变化，不会急躁、焦虑或者出现心理波动，所以电脑对于盘面招数的变化是反应既快又不会犯错误，人脑根本无法与之匹敌。20世纪90年代末，一些象棋特级大师跟计算机下棋还是胜负各半，进入21世纪，即使是许银川、吕钦这样的特级大师，想在电脑软件身上占便宜都已经不可能了。电脑软件发展到今天，特级大师与其对阵根本没有赢棋的把握。
在回顾象棋软件发展的同时，作者指出象棋软件对于象棋运动整体发展的好处：一是棋迷朋友们可以在软件中把难度设置成适合自己的级别，用于练棋；二是可以软件打谱，帮助人分析每个局面下的所有应对方式，从头到尾、全面系统地把一盘棋彻底研究透。

阅读收获

通过阅读该书我认识到江湖棋局都是深深的套路，业余棋迷们万不可陷入其中。让子棋有诸多的规矩，应该超出了大多数爱好者的认知范围。棋力再高的人也会有短板弱项，比如情绪的把控、思维方式、性格，甚至生活习惯等等。
近几年，我心中逐渐萌生写作的想法，把自己这些年来的故事和回忆写下来，形成文字。作为一名看到街边象棋棋摊儿就走不动路的爱好者， 阅读这本书的过程，再次点燃了我写作象棋话题的热情。
