大学生活

不知不觉间参加工作已十年有余，如今回望四年本科生活，来侃一下至今仍然留驻脑海的苦与乐。
先说值得自己骄傲的，共有*件事情。按时间发生顺序，首先，
大一下期末《电路基础》考试中，我们学院全年级共有四名同学得100分，我是其中之一。教我们的王伞老师请我们四个考满分的同学以及“课代表”馅儿在学子餐厅吃大餐，吃饭的过程中王老师自嘲说，我发现一个事实，你们这几个拿满分的同学都没有参加我的考前辅导答疑课。其他几位同学我不知道，考前辅导我确实没去，因为我备考时把通读复习了教材，配套的习题册上的所有题目刷了一遍，然后再回看一遍教材，感觉对电路基础课程中的阶跃响应、冲击响应、卷积等知识体系有了深刻的理解，形成一种对于考试题来者不拒的强大气场。这一优异成绩不单赢了来同学们羡慕的目光，还助我拿到了二等奖学金。谁能想到呢？工作后，我参加2014年研究生入学考试时，业务课2依然是《电路基础》，备考时我没有复习基础教材，直接开学推荐的《重点难点考点辅导与精析》与《电路基础常见题型解析及模拟题（第3版）》两本参考书，考试中只有一道大题确实不会做，最终取得了135分的成绩。
也就是在电路基础考试后的那个寒假，回郑州时我和向阳同学同行，聊到买返校车票的事儿，他说他有个亲戚在郑州，可以帮忙买票，于是我把学生给他了，让到时候帮也给我买一张。结果，买到的车票比开学时间早了半个月，大概初八左右，我就早早的回到了学校。同宿舍的也有两三个要补考的同学提前到校了，他们忙于复习，我则无所事事，在大个的电脑上尝试着学玩“魔兽争霸”的游戏（以下简称War3）。
宿舍八个人中，大个、老张他们玩已经玩了很长时间，算是这个游戏老手了，舍长是自大学以来才开始玩的，算是后起之秀。《冰封王座十大经典战役》解说视频火爆的时候，他们想追剧一样痴迷，每当有新视频发布时，有大个下载到他自己的电脑，然后他们几个一遍又一遍单曲循环。《经典战役》选取了War3这款即时战略游戏中颇具代表性的十个战例进行讲解，女解说“稻草芭比”的声音磁性十足，极富激情而又不失专业性，耳染目濡之下，我改变了对这款游戏的态度，渐渐地对这款游戏有产生了好感。
这一尝试，恰如打开了潘多拉魔盒。在他们复习准备补考的空档儿，我从对战简单电脑玩家开始，一次次的练习，升级到中等，继而又打败了疯狂电脑级别。之后开始和舍友对战，一开始他们让我50%血（游戏中所有单位生命值只有默认的一半），输了之后，以每10%一个台阶增加，直至我赢过宿舍内其他人80%血的水平。不服输的我在War3的路上越走越远，后来对这款游戏上瘾了，达到了痴迷的状态。借口学习需要，跟家里要了将近6000元，在舍友老张、大个等的陪同下到船舶电子大世界攒了一台当时来看主流配置的台式电脑。加上我的电脑，宿舍中共有四台电脑，为联机对战创造了极佳的条件，正值青春年少的我们谁也不服输，轮番上阵。扬言说谁谁谁不是个，发出赤裸裸的挑战。一旦输了，八成要找时机赢回来。那时候，LostTemple Turturock就是我们的世界地图。最疯狂的时候，12:00下课，吃过午饭就急匆匆地跑回宿舍，赶在下午上课前搞上两局。有些时候，两个人较起真儿来下午的选修课也就翘掉了，一直对战到筋疲力尽方肯罢休。更有甚者，整个宿舍全员出动到学校附近的网吧里面包夜，玩通宵。
与大多数舍友不同，他们只当是游戏，我不仅会保存自己的游戏录像，及时回放观看，找出自己不足的地方，以便下次玩得更好；还观看网络上大小赛事的直播，下载第一人称视角的教学录像，用专门的微操地图来辅助练习，颇有些研究的意思。到后来舍友们陆续开始谈对象，游戏逐渐让位于恋爱，对War3的热情慢慢退去时，，我通过大量的实战和复盘在宿舍内已达到数一数二的水平，与他们中的任何人对战时已有对半甚至更高的胜率。 （//：呸，也不想想，你自己在浩方、VS等对战平台上消耗了多少青春年华，别人收获了宝贵的恋爱经验，你呢？）

大四时，我们搬到新的宿舍楼，辅导员对我们的管理也愈来愈严格起来，要求我们把电脑集中到公寓楼一层的学习室内集中存放，限定开放的时间。我极不情愿，就把电脑藏到柜子里面，偷偷地玩。有一次被兼职的辅导员这个正着，他批评我说，你还考不考研？天天这么捉迷藏似的疯玩，能考上吗？我当时玩得正在兴头上，没把他说的太当回事儿，说了“我考不考得上跟你有啥关系”之类的话。当天晚上，我主动去找他承认错误，为自己的恶劣态度向他道歉时，他继续教育道，“二十四史”你看过吗？你们这些学生，心里就知道玩游戏，多到图书馆读读书多好啊。自此以后，我玩War3的频率才慢慢降下来，从一天玩两三局到一天一局，然后两三天才打一局，再然后一个星期最多打上两局。

以当时的眼光来看，War3也带给我不少的“好”处，比如说与舍友之间的关系更为融洽，因这个话题与班上其他游戏的数名同学有了更多的交集等。这中间值得一提的只有一件事儿。
在**学期的金工实习期间，有一个环节是数控机床方面的，课程作业要求编写一段类似于汇编的代码用于控制机床的垂直的切割线，从厚度为1mm的不锈钢板上切下自己设计的平面图形。数控机床只接受线段和圆弧两种指令，线段需要指定起点和重点的坐标，圆弧需要指定圆心坐标、半径和弧度数。每名同学可以任选图形，交作业时需提供相应的控制代码。
同宿舍的舍长同学选择了一个船锚的图形，直线加圆弧约十段左右。我选择的图形是War3中人族的徽章，。为了将这个图形转换为代码，我先从宿舍大个电脑上拷贝图片后，到打印店和老板协商能不能帮忙把这个图片打印到坐标纸上。老板一听是课程作业，拿来一看，态度极其友善地说，你这个背景的横条应该是不要的吧？我说，对对对。然后老板花了十几分钟的时间，帮我处理好了图片，只留下轮廓部分，打印在坐标纸上。然后我姓高彩烈地找了个自习室，用圆规直尺一点点地测量、比对、计算，找出图中每一个转折点的坐标，以及每一段圆弧的圆心、半径、度数。整整一个晚自习的时间我都乐此不疲，理清图形中68条之多的直线和圆弧，写下的控制代码铺满了两页B5纸。
没想到的是，我的作品是班内所有同学中最复杂的一个。最后一次课时，老师共选五副作品在机床上加工，我的作品赫然在列。加工时放了五层钢板，完成后，有三五位同学表示想留一个做纪念，我送出去了三个，自己留了两个。
