Hello Go361

“每一个人就是一个世界，这个世界是随他而生，随他而灭的。每一块墓碑下面，都躺着一部整个世界的历史。”
[——亨利•海涅]]

近来在微信公众号平台推送了几篇文章，写的是生活中的一些事情和感悟，文中的想法和观点还不很成熟，记录下来主要是作为自己成长过程的一个见证。
有朋友反馈说“看完你的文章后不知道你想表达什么，感觉文章没有围绕一个点展开。”这几天反复琢磨后感觉应该补个开篇，阐明自己写作的初衷、想要达到的目的，后续文章会围绕哪些主题展开等，以使关注的读者有适当的预期。

写作的初衷

钱穆曾在80岁高龄，历时5年著述《八十忆双亲·师友杂忆》，书的序言中写道“既到老不忘，则可确证其为余生命中之重要部分，务求叙述真实，亦属余对生命之自惜。”2010年10月，特殊的心境下，我拜读大师的这一著作，感触良多，心中萌生出系统地回顾自己成长经历的想法。（试想50年后，回顾一生时会是什么情景呢？）在这一想法的促使下，我断续地写了数十篇文章，但每次重读时都对文章的质量不甚满意，数次易稿，鲜有发表。
2016年3月阅读书籍《奇特的一生》后，我开始实践其中的时间记录法，三年以来行动量较前些年有幅度的增长，主要集中在读书写作、学英语、编程（Python）等几个方面。随着行动量的增加，我在微信公众号先后起草了二十多篇文字。
2017年12月重温“兰迪 波许教授的最后一课”演讲时，糖豆已近三周岁，思考自己当下的生活，脑海中不止一次地设想如下场景:儿女在今后成长过程中遇到困惑，问:“爸，您像我这么大的时候是什么样子，都在做什么？”我该作何回答呢？通过写作来记录自己的成长过程应该是比较理想的办法。
2018年底写给自己的总结中，我也曾再次确认写作是自我成长道路上必不可少、至关重要的一个环节。
追寻十年来思维的变化及行动踪迹，写作路上的我，几度雄心壮志，几度无疾而终，致使自己的认知、思考水平至今仍在低层次徘徊，无实质性进步。今年四月起，综合读书过程中的一些体会，我决定重新走上写作之路，旨在剖析自我，回顾成长经历，探索发展方向。

后续文章选题

后续文章计划分为旧的回忆、活在当下、善假于物、破万卷书、以及扶轮问路五个方面的内容，具体如下。
旧的回忆( Memory )：回顾自己的成长历程；
活在当下( Present )：日常生活中的随想与感悟；
善假于物( Tools )：使用电脑手机软件的经验；
破万卷书( Reading )：读过的一些书及心得体会；
扶轮问路( Future )：对于未来的思考与规划。
上述选题只是一个初步的构想，其实自己还没有做出什么成绩，写作这些选题主要是为了回顾求学、工作以及生活中经历的一些事情，总结经验教训。望自己能做到持续行动，笔耕不辍。先定个小目标，争取早日写到100篇。
如果这些文章能勾起读者心中美好的回忆，或者对读者成长过程有些许的参考借鉴意义，那将是我最大的荣幸。同时也欢迎前辈们、朋友们能提出宝贵的指导意见。
