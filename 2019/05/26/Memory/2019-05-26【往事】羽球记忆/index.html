Memory

趁着中国队再捧苏杯的喜庆，聊聊我打羽毛球的故事

##结缘，萌芽时期（—11.2）
幼时曾在外婆家和伯父家见过羽毛球拍和羽毛球，那个时候羽毛球于我只能算是一种玩具。
高一时，教我们英语的老师体育特长是羽毛球，班上的同学曾和他切磋过，虽然没有网子和标准场地，但依然能感觉到老师的水平高过我们好几个层级。
大一寒假期间去县城逛街，在姐姐的赞助下买了我们家里的第一副羽毛球拍，当时大概只买了三五个羽毛球，我开学后那副羽毛球拍基本上也就退役了。
大二体育课，有部分舍友选择了羽毛球项目，我还是保守地选择了熟悉的乒乓球。后来舍友们聊起来说结业考核要求“把羽毛球发到场地的固定区域”，我心里还顿生疑惑“这不可能做到吧”。
2010年3月前往西安参加为期十个月的培训班，宿舍楼下面恰好有两片露天羽毛球场地，下午上完课回到宿舍后，一闲下来就从窗台上趴着看下面的人打球。培训班上有个叫韩磊的高手，经常带着班里的几个同学在楼下打球；场地上偶尔能看到一个十分厉害的老师，充不多能单挑两个韩磊。看得多了，渐渐发现羽毛球是一项非常有趣的运动——跑动范围比乒乓球大得多、没有那种令人捉摸不定的旋转变化，没有篮球那种激烈的身体对抗，力量、速度、落点、线路等变化十分丰富。
在耳染目濡的熏陶下，我在5月12日—16日期间破天荒地认真观看了电视直播的汤尤杯比赛，8月份逐渐开始和同学们一起打羽毛球，参与的次数并不多，连基本的握拍姿势都没有很好地掌握，但正是这段弥足珍贵的经历开启了我迈入羽毛球殿堂的大门。
##痴迷，飞速成长(11-14)
2011年初回到工作单位后，单位里面正好掀起了打羽毛球的热潮，在达哥、豪哥、贺总的带领下，同事们先后购买了价值数百元的球拍，开始学打羽毛球。我于2月10日起逐渐融入打羽毛球的圈子，形成打球习惯后，购买了第一把球拍（Victor nano 9），日渐痴迷于羽毛球运动。自此踏上不归路，羽毛球成为业余体育运动的首选项目。
有记录可查的打球次数：
2011年上半年打球共计46次，
2012年共计打球83次，
2013年上半年共计打羽毛球77次
比赛直播及录像
实战之余还观看肖杰、赵建华教学视频《专家把脉》，
11年5月22日至29日期间第一次观看了苏迪曼杯混合团体锦标赛，12月中旬从达哥那儿拷贝大量羽毛球比赛视频，基本上每个看过两遍以上。
2012年全程观看奥运会羽毛球比赛直播
装备选购
2011年4月8日4月8号购买了自己的第一把球拍（Victor nano 9）；
2012年选购第一双羽毛球鞋（凯胜），8月选购自己的第一套羽毛球服（Victor）；
2013年10月第二双羽毛球鞋（YY SHB-45C 海军蓝 ￥343）、Yonex羽毛球袜
###球友及与外界的交流
阿达是我们中多数人的启蒙老师，曾经跟着专业的老师学过球，从握拍、高远球、吊球、杀球、平抽、勾对角等一个个动作地教会我们打球。其爆发力很好，动作标准，杀球威力大，封网扑杀杀伤力都非常强。
贺总体能极好，跑动能力十分突出，学习半年多之后以逐渐感觉不到他在单项技术上的优势。
标哥是属于我们单位的体育健将，是篮球、足球、乒乓球、羽毛球四个项目都可以排入第一集团的牛人。跟他打球就一个字“累”，能把人跑死。
豪哥是乒乓球准专业选手，曾打进市里面前三名。令人着实佩服的是他右手握拍打乒乓球，羽毛球项目上则是左手握拍，技术也相当细腻。
老魏是我培训结束回到单位时的舍友，打球出勤率极高，杀球能力比较突出，步法上相对弱一些。
房子也是左手握拍的一名选手，出勤率比我们几个低一些，但是球也大的相当不错，身上有一种永不服输、拼到最后一分的倔强劲儿。
雷帅是我们中起步较晚的一个，大概是从12年开始加入打球的团队，但借着身体素质上的优势，他用了不到一年的时间就基本上追平了我们的水平。平板支撑20分钟+的身体素质着实令人羡慕。
11年买了球拍的部分同事后来没有在羽毛球项目上继续投入更多的精力，这里就不一一列举名字了。
其中到黄冈市区与其他球友交流共两次，工作单位所在地的高中老师到我们单位交流一次。
##平稳期，走出去(14.9-17.5)
2014年4月参加研究生复试期间，曾逛到学校的体育馆观摩羽毛球教学，感受学校的羽毛球氛围。9月份入学之后，刚加入年级群就在里面询问有没有打羽毛球的同学，第一次到体育馆打球认识了李鑫、李明、童淑君等同班同学，第二次去打球有幸被学院羽毛球队选中成为其中的一员。随后在体育文化月学院组织了班级羽毛球团体赛，带领班级进入年级前三名。
研究生期间除了参加学院羽毛球的活动，和学院球队的伙伴一起参加“三航杯”，还参加过一个业余比赛。学院队的圈子之外，最多的就是和李鑫打球了，由于学校羽毛球场地紧张，经常两个人趁中午没人的时候到体育馆刷卡打球。后来还在李的推荐下认识了器材党土豪黄柏力，印象中共和黄打过三次球（两次在东大，一次在老校区）。
15年3月购买自己的第二把球拍YY（NR-ZSP），
15年11月，尤尼克斯 SHB-45C 橙黑 鞋码: 42 （￥282）
16年购买第二把球拍Victor（极速12）
