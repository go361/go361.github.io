Reading

24年1月13日，在黄冈市图书馆发现了吴军老师的《见识》，连续五天内用时大约8小时读完了全书。

什么是命呢？不同人有不同的理解。对我来讲，它主要取决于两个因素，环境的因素和我们自己对未来走向划定的方向。环境因素不能忽视，第二个因素则掌握在每个人自己手里。所谓命就是一个人看问题和做事情的方法——习惯、行为及其背后的思维方式。

##Ch1 幸福是目的，成功是手段

幸福感的本源只有两个——基因的传承和影响力。（P005）

这个世界没有欠你什么

寒窗苦读知识一种读书态度，这种态度是好的，但是社会竞争是一种非常复杂的长期系统性的竞赛，寒窗苦读是成功的因素之一，和经济条件好、智商高、出身好、长得美一样，都只是其中的一个变量而已，而命运是多个变量互动的结果。（P008）

我们必须承认，任何社会都是分层次的。事实上，在任何国家、任何时代，社会都是分层的。稍微好一点的社会不过是有一个上下层之间的通道，让人员可以流动而已。

庆幸的是，绝大部分人在一生中的大部分时间是处在松懈状态，而达不到80%的努力程度，这才给了逆袭者以机会。不过，处在第10层的人可能只需要付出10%的努力，产生的结果都比那些第70层的人付出100%的努力结果要更好。
对于那些试图在金字塔上爬几个台阶的人来讲，要做的不是抱怨社会的不公平，而是需要付出足够的努力，同时把注意力放到最该关注的事情上去。虽然我们通常会抱怨社会阶层固化，但是往下的通道永远是非常宽的，只要稍微不努力或多抱怨几句，就能往下走几层；相反，往上的通道即使再宽，往上走也是一件辛苦的事情，如果不能看到自己的不足，并且用半辈子的时间来弥补，又用半辈子的时间往上走，那么能够维持现有的阶层已经是烧高香了。

人生最重要的投资

对于年轻人来讲，对自己的投资和在职业上的进步，远比在股市上捞点钱或者向父母借钱买一套房子更为重要，也更为靠得住。当然，还有一个和投资自己至少同样重要甚至更重要的投资，就是找一个好的配偶。

先让父母成熟起来

只有父母成熟了，才能让孩子有一个好的起点。
父母成熟，首先自己要不断地进步。除了与时俱进和给孩子做好表率之外，中国这一两代父母还要完成一个非常艰巨的任务，即在观念上从农耕文明的思维方式向现代商业文明的思维方式转变。

年轻人应该把自己看成是一个成年人，用一种尊敬，而且是成年人之间的沟通方式和长辈说话，要不断囧自己接受的新思维方式和理念传递给父母。

##睡眠（1096h）
根据往年统计平均每天睡眠为 7.5小时，理应养成良好的作息习惯，并持之以恒地执行，力争避免因睡眠时长波动导致的工作学习效率下降。
理想作息：22:30卧床入睡；05:00起床，早上阅读1h；中午12:45-13:45午休约1小时。
6月9日，因B站上的ChatGPT视频，在渐构个人网站上学习了关于睡眠的一个视频，总结要点是：

  • 持续规律健康的睡眠会给人带来长期收益；
  • 早晨和傍晚感受阳光，启动并校准体内生物钟；
  • 20:00后，尤其是23:00~04:00之间，不要让眼睛暴露在明亮的光照下；
  • 午睡，建议时间控制在20~30分钟，一个小时以内（非深度睡眠）；
  • 辩证看待大脑与身体的关系，通过控制身体以控制大脑。

弱人择原理：
在一个强大的或具有无限空间和/或时间的宇宙里，只有在时空的有限的一定区域里，才存在智慧生命发展的必要条件。在这些区域中，如果智慧生物观察到他们在宇宙中的位置满足那些他们生存所需要的条件，他们就不应该感到惊讶。
强人择原理：
存在许多不同宇宙或者一个宇宙的许多不同区域，每一个都有自己的初始结构，或许还有自己的一套科学定律。在这些大多数宇宙中，不具备复杂基体发展的条件；只有在像我们的很少的一些宇宙中，智慧生命才得以发展并能质疑“为何宇宙是我们看到的这种样子？”回答很简单：如果它不是这个样子，我们就不会在这里。

收获与进步？

缺乏独立思考：
《活出人生的意义》
只有经过独立思考，理论联系实际，才能够把学到的知识转化为自己的知识，才能学以致用。

人与人的头脑本来就存在着差异，这种差异使有些人喜欢读书，有些人更倾向于独立思考。

读书不能等同于独立思考，它只是对我们的思考起到一种引导的作用。……最有效的阅读应该在自己的思想源泉枯竭时才去读书。

单纯的读书其实是在读取别人的思考成果。自己独立思考出来的东西虽然不见得是一个完美的知识体系，但也是一个首尾一致的总体。如果没有自己的独立思考，总是单纯地从书本中吸取别人的思想，你就会变成一个支离破碎的人，没有主观，不能形成正确的判断力，甚至精神紊乱。

阅读是思考的代替品，它呈现给我们的是别人的思考成果，为我们自己的思考提供素材，读书的作用仅此而已，不要读太多书而让它喧宾夺主，代替了我们的独立思考，使我们失去对客观生活的关注，习惯于走别人的老路。

[221108【思索】202205-202210]

对比《教子有方》中讲解志向一章，反思过往——大学毕业后按定向分配就业至今已十四年有余，发现自己大多数年月里都处于一种随遇而安的状态，甚至于有些阶段连一个明确的小目标都没有，更谈不上什么远大志向和理想了。这样的生活状态导致了一个很严重的问题，那就是整个人要么处于一种无所事事的信步漫游状态，要么处于另一种三分钟热度的打鸡血状态。在前一种状态里，整个人体现出一种颓废感，生活了无生趣，找不到人生的意义；在后一种状态中，往往是为了生活、工作中的紧急事务忙忙碌碌，或者为了一时兴起的想法奋起直追。两种状态形成了某种循环往复的周期规律，大体上随着工作的紧张程度起伏变化。颓废期间的典型生活表现是消耗很多时间精力用于刷影视剧、打游戏、报复性地一局又一局地下没有思考深度的棋等，时间久了，内心向善的一面会琢磨着要找点儿有意义的事儿做；鸡血状态中往往是毫无作息规律地忙于临时事务或短期目标，过不了多久，体力精力消耗殆尽，内心恶魔的一面怂恿着说即便那么拼命也改变不了命运（做不到阶层的跃迁），还不如佛系一些，适当放纵一下自我。这两种状态都没有很好地把握住度的平衡，结果只能是在同一层级上反复螺旋，实质上没有上升。

最大的收获：平和的心态去对待工作生活中事情。

如何离线安装python包

b、下载安装对应版本的.whl压缩包
前往pip官方源https://pypi.org/或国内pip源 https://pypi.tuna.tsinghua.edu.cn/simple/ 进行手动下载对应包的.whl文件，这里推荐使用国内pip源（后者）进行下载，下载速度较快。
由于whl是深度学习依赖包对应的压缩包，在其中包含了py文件，以及经过编译的pyd文件，因此可以直接进行安装，安装命令为：

pip install *.whl

c、下载安装.tar.gz或.zip源码包
在pypi的官网或者国内源pypi包中有些并不包含whl压缩包，但一般会有.tar.gz或者.zip的源码包，对于下载的源码包需要通过解压后得到的setup.py进行编译再安装，编译的命令为：

python setup.py build

安装的命令为：

python setup.py install

自11月6日隔离生活起，开启了新一段的读书旅程。原以是为隔离三天，只准备了《中国近代史新编》和《浮生六记》两本十月份没读完的书，进度分别为50%和35%；实际上隔离了八天，读完《中国近代史新编》后，利用相对集中的自由时间读完《认知心理学》，又开启了《一本书读懂台湾史》的阅读。结束隔离时，单位年终检查已经开始了，这意味着工作进入相对轻松的阶段。

隔离之后，连续两个周末都处于封闭管理的静默状态，19日早晨醒得比较早，05:00时洗漱好开始阅读《中国思想史》一书，接下来连续五天内用时13小时读完。正如2019年读《中国通史》的感觉，春秋时代的百家争鸣是中国思想史上最璀璨的篇章。全书读下来，除了春秋战国时代之外，印象最深的要属书目收尾处讲解“孙中山”的一节。

严格来讲，近五十年来，中国亦并无所谓思想界。只有孙中山一人，他终身从事革命的实际工作，固不该专以思想家目之。但中山先生实有他独特的一套思想，他不仅堪当这一百年来近代中国唯一的一个思想家，而且无疑地他仍将是此后中国思想新生首先一个领导人。
马克思以物质为历史的重心是不对的。社会问题才是历史的重心。
人类求生存，才是社会进化的原因。阶级斗争，不是社会进化的原因，而是社会当进化时发生的一种病症。
孙中山的《三民主义》，应该是近代中国新生唯一的启示。若我们把一个人的生命作比喻，民族主义是其人之精神与灵魂，民权主义是骨骼，民生主义是血肉。三者缺一不可。

读完《中国思想史》后，又相继在三天内用时9个小时阅读了钱穆大师的《人生十论》，这本书读来是自己的内心趋于平和，与今年七月至十月间的急躁形成鲜明对比。
钱穆认为，人生是“事业”与“性情”的一体两面，奉劝大家，“莫要太看重了外面事业，而忽略了内部性情。”

性情不是天生的，需注意后天培养。从个人言，各该当心不断自修自养。从大群言，我们要认识中国人性情来培养我们自己的性情，从中国的文学与艺术中多所采撷，来帮助自己作人生的修养。
