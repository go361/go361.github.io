七牛云空间文件迁移

七牛云真是恶心的不行，测试域名失效之后，不单是外链失效，即便进入到自己的空间，也不能下载相应的资源。
捣鼓了将近三个小时，终于解决了——测试域名已失效的七牛云空间中文件迁移这个问题。

解决思路

参考知乎专栏文章七牛云测试域名到期失效，下载空间文件办法，这个讲的比较细致。
另外，也可以参考七牛云域名失效后如何下载文件这篇文章，原理是一样的，只不过这个文章讲的是linux下的操作。其中获取第一列 文件名列表的操作 真的很简洁优雅。

操作步骤

1. 在七牛云控制台新建空间

浏览器进入七牛云控制台，对象存储中新建一个空间，用来把域名失效空间内的文件拷贝进去。因为新建空间可使用测试域名，所以可以从新建空间下载原来的文件，曲线救国。

注意：新建空间一定要与域名失效空间在一个地区服务器上，否则无法拷贝进文件。

2. 安装七牛云官方命令行工具 qshell

从以下地址下载工具包：点击下载
也可以在github上下载，github上qshell的文档

3. 七牛账户密钥设置

使用 qshell 操作七牛空间，需要添加账户密钥 AccessKey 和 SecretKey, 赋予 qshell 权限。

这一步中，我的AK和SK，测试了很多遍都提示bad token，最终在七牛云设置中 新创建了一个密钥，然后停用了旧的密钥。（可能是因为太久没有使用的关系？上次使用是三年以前的事情了。）

4. 转移失效域名空间文件至新建空间

qshell.exe batchcopy
假设失效域名空间名叫 aaa，新建空间名叫 bbb。则从 aaa 批量拷贝文件到 bbb 命令如下：

在当前工作目录中便出现 tocopy.txt，再用上面的批量拷贝命令，便能将 aaa 中文件全部拷贝进 bbb 中。

这里需要注意：导出的文件列表中 还有除了文件名之外的多余字段，导致直接使用该txt列表去执行 batchcop 命令时，提示：** Failed, Code: 612, Error: no such file or directory**

去除 多余字段的解决办法：
将txt中的内容全选，粘贴至 excel，excel会自动识别tab 分隔符，第一列即为文件名。

选择第一列，复制后粘贴至新的txt文档，比如list.txt;
以list.txt为参数，执行batchcop命令。

拷贝后可以到七牛云的对象存储中去浏览一下。

5. 从新建空间中下载所有文件

用如下命令将新建空间 bbb 中的所有文件同步至本地目录：

./qshell.exe qdownload download.conf
其中 download.conf 为下载配置文件，在工作目录中创建，内容按如下填写：

{
“dest_dir” : “本地下载目录绝对路径”,
“bucket” : “要下载文件的空间名”,
“cdn_domain” : “新建空间的测试域名”
}

注意事项：

  • 在Windows系统下面使用的时候，注意dest_dir的设置遵循D:\jemy\backup这种方式。也就是路径里面的\要有两个（\）。
  • “cdn_domain”如果不配置为测试域名，则 qshell 下载文件不走 CDN 免费流量，需支付流量费。
    执行下载命令后，稍等片刻，便能在本地下载目录中看到失而复得的文件。

6. 再上传到靠谱的新图床

因为文件名没变，批量修改博客中图片外链的前缀图床地址，图片就能恢复显示。
